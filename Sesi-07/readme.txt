apa itu web/software testing ? 
Web testing adalah proses pengujian aplikasi web untuk memastikan bahwa aplikasi tersebut berjalan dengan baik dan sesuai dengan yang diharapkan. 
Web testing juga bertujuan untuk memastikan bahwa aplikasi web tersebut dapat berjalan dengan baik di berbagai browser dan sistem operasi yang berbeda.

yang kita akan gunakan adalah :
- Unit Testing
- Functional Testing
- Integration Testing
- Manual Testing

- Unit Testing mengacu pada pengujian yang dilakukan pada unit kode. 
Unit kode adalah unit terkecil dari kode yang dapat diuji secara terpisah. 
Biasanya, unit kode adalah fungsi atau metode. 
Unit testing dilakukan selama proses pengembangan aplikasi oleh pengembang. 
Unit testing dilakukan dengan menggunakan kerangka kerja pengujian unit seperti JUnit, NUnit, dll.
- Functional Testing mengacu pada pengujian yang dilakukan untuk memastikan bahwa fitur aplikasi berfungsi dengan benar.
- Integration Testing mengacu pada pengujian yang dilakukan untuk memastikan bahwa beberapa unit aplikasi bekerja bersama dengan baik.
- Manual Testing mengacu pada pengujian yang dilakukan secara manual oleh manusia.
