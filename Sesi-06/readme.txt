Apa itu dom (document objek model) dan bagaimana cara menggunakannya?
DOM adalah sebuah API (Application Programming Interface) untuk dokumen HTML dan XML. 
DOM menyediakan representasi struktur dokumen sebagai node-node dan objek-objek yang dapat diakses dan dimanipulasi. 
DOM menyediakan cara untuk mengakses dan mengubah isi, struktur, dan gaya dari sebuah dokumen. 
DOM menyediakan representasi dokumen sebagai node-node dan objek-objek yang dapat diakses dan dimanipulasi. 

Apa itu regex dan bagaimana cara menggunakannya?
Regex adalah singkatan dari Regular Expression, adalah sebuah metode untuk mencocokan pola dalam sebuah string.
Regex digunakan dalam bahasa pemrograman seperti Java, Javascript, PHP, Phyton, C dll.
Regex juga digunakan dalam aplikasi pengolah teks seperti grep, sed, dan awk.
